-- Property Biding Exemple --

Below I´ve created what it´s called "property binding" on angular. Insted of using the href tag on anchor html element, i´ve declared a variable inside the page component with the youtube course url and passed it to the [href]="variableName" that is the syntax to property bindind. You can use property bindin on any attribute of html elements as you wish to bing their value to a variable declared as a propoerty inside the component class.
